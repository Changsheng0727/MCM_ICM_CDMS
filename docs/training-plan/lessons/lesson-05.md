# 第 5 讲：回归模型与拟合诊断

## 课程定位

这一讲对应 `m_Sai_03_1.mlx` 中的回归部分，目标是让学员会建立并解释回归模型。

## 课堂目标

- 建立一元和多元线性回归
- 理解回归系数、拟合优度和残差
- 输出拟合图和结果表
- 写出论文式结果解释

## 核心概念

- 自变量与因变量
- 最小二乘
- 回归系数
- 残差
- 模型诊断

## 课堂演示

基于糖尿病数据演示：

- 葡萄糖与年龄、BMI 等变量的关系
- 一元回归和多元回归
- 拟合图与残差图

## 课堂练习

让学员选择 2-3 个自变量建立回归模型，并比较模型效果。

## 课后作业

提交：

- 回归代码
- 回归系数表
- 拟合图和残差图
- 150-250 字结果解释

## 验收标准

| 项目 | 标准 |
| --- | --- |
| 变量选择 | 自变量和因变量说明清楚 |
| 代码 | 模型可运行 |
| 诊断 | 至少包含残差分析 |
| 解释 | 能说明结果意义和局限 |

## 后续建设建议

将回归部分从 `m_Sai_03_1.mlx` 拆成独立脚本，并保存系数表。
