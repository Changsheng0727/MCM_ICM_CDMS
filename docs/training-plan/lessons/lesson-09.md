# 第 9 讲：分类模型与机器学习入门

## 课程定位

这一讲让学员具备处理有标签分类问题的基本能力。

## 课堂目标

- 区分回归、分类和聚类
- 建立基础分类模型
- 使用训练集和测试集验证模型
- 理解混淆矩阵、准确率、召回率等指标

## 核心概念

- 标签
- 特征
- 训练集和测试集
- 混淆矩阵
- 过拟合

## 课堂演示

使用二分类数据演示：

- 数据划分
- 逻辑回归或决策树
- 混淆矩阵
- 指标解释

## 课堂练习

学员建立一个基础分类模型，并解释错误分类样本可能来自哪里。

## 课后作业

提交：

- 分类模型代码
- 混淆矩阵
- 指标表
- 误判样本分析

## 验收标准

| 项目 | 标准 |
| --- | --- |
| 划分 | 训练和测试分开 |
| 指标 | 指标解释正确 |
| 解释 | 能说明模型不足 |
| 复现 | 结果可重复 |

## 后续建设建议

补充随机森林和特征重要性示例。
