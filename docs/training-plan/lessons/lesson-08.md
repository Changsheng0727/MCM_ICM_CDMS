# 第 8 讲：预测模型基础

## 课程定位

这一讲补齐预测类问题，帮助学员从历史数据推断未来趋势。

## 课堂目标

- 理解预测问题的数据要求
- 掌握趋势外推、回归预测和灰色预测
- 理解训练集、测试集和误差评价
- 输出预测曲线和误差表

## 核心概念

- 时间序列
- 趋势外推
- 灰色预测
- 误差指标
- 预测区间

## 课堂演示

使用一份年度或月度数据演示：

- 趋势可视化
- 简单预测模型
- 误差计算
- 预测结果解释

## 课堂练习

学员对一组时间序列数据做短期预测，并比较两种方法。

## 课后作业

提交：

- 预测代码
- 预测曲线
- 误差表
- 预测结论和风险说明

## 验收标准

| 项目 | 标准 |
| --- | --- |
| 数据划分 | 训练和验证逻辑清楚 |
| 模型 | 预测方法适配数据 |
| 误差 | 至少使用 1 个误差指标 |
| 解释 | 不夸大预测可靠性 |

## 后续建设建议

补充 ARIMA 或指数平滑作为进阶内容。
