# 第 12 讲：图论与网络模型

## 课程定位

这一讲覆盖交通、物流、传播、关系网络等题型中的常用网络模型。

## 课堂目标

- 将实际系统抽象为节点和边
- 掌握最短路、最大流和网络指标
- 能画出网络并解释关键节点
- 能把网络结果转成策略建议

## 核心概念

- 图
- 节点和边
- 最短路径
- 最大流
- 中心性

## 课堂演示

使用一个小型交通或物流网络演示：

- 构建邻接矩阵
- 求最短路
- 计算关键节点指标
- 可视化网络

## 课堂练习

学员对一个小网络完成最短路和节点重要性分析。

## 课后作业

提交：

- 网络建模说明
- MATLAB 网络代码
- 网络图
- 路径或节点重要性结果

## 验收标准

| 项目 | 标准 |
| --- | --- |
| 抽象 | 节点和边定义合理 |
| 算法 | 模型与问题匹配 |
| 可视化 | 网络图可读 |
| 解释 | 结果能转成建议 |

## 后续建设建议

新增物流配送或传播网络案例。
