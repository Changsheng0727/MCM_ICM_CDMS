# 第 11 讲：整数规划、0-1 规划与多目标优化

## 课程定位

这一讲承接线性规划，处理离散选择、开关决策和多目标权衡。

## 课堂目标

- 理解整数变量和 0-1 变量
- 能用枚举、小规模搜索或 MILP 工具求解基础问题
- 理解多目标优化的加权求和思想
- 能比较不同权重下的解

## 核心概念

- 整数规划
- 0-1 规划
- 多目标优化
- 权重
- Pareto 思想

## 课堂演示

演示：

- 选址或任务分配问题
- 0-1 变量建模
- 多目标加权对比

## 课堂练习

学员完成一个“选或不选”的优化问题，并说明整数约束带来的变化。

## 课后作业

提交：

- 整数规划模型
- 求解代码
- 不同权重的结果对比
- 结果解释

## 验收标准

| 项目 | 标准 |
| --- | --- |
| 变量 | 整数或 0-1 变量定义正确 |
| 约束 | 逻辑约束完整 |
| 权衡 | 多目标解释清楚 |
| 结果 | 解可复现 |
