# 第 14 讲：敏感性、鲁棒性与模型检验

## 课程定位

这一讲解决论文中最容易薄弱的部分：模型是否可信。

## 课堂目标

- 理解敏感性分析、鲁棒性分析和模型检验的区别
- 能改变关键参数观察结果变化
- 能用验证集或对比模型检验结果
- 能写出可靠性说明

## 核心概念

- 敏感性分析
- 鲁棒性
- 验证
- 对比实验
- 误差来源

## 课堂演示

选取前面某个回归、评价或优化模型，演示：

- 改变关键参数
- 比较输出变化
- 绘制敏感性曲线
- 写可靠性段落

## 课堂练习

学员对自己的前期作业做一次敏感性分析。

## 课后作业

提交：

- 参数扰动设置
- 敏感性图
- 鲁棒性结论
- 模型局限说明

## 验收标准

| 项目 | 标准 |
| --- | --- |
| 参数 | 选择关键参数 |
| 分析 | 结果变化有图表 |
| 结论 | 能说明模型可靠范围 |
| 局限 | 不回避模型问题 |
