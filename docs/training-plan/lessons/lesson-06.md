# 第 6 讲：聚类分析与 PCA 降维

## 课程定位

这一讲对应 `m_Sai_03_1.mlx` 中的聚类和 PCA 部分，目标是训练无监督分析能力。

## 课堂目标

- 理解无监督学习适用场景
- 掌握 K-means 聚类基本流程
- 理解特征标准化的重要性
- 用 PCA 可视化高维聚类结果

## 核心概念

- K-means
- 标准化
- K 值选择
- PCA
- 聚类画像

## 课堂演示

基于糖尿病数据演示：

- 选择聚类特征
- 标准化
- 肘部法选择 K
- PCA 可视化聚类结果
- 计算每类特征均值

## 课堂练习

学员调整特征组合和 K 值，比较聚类结果是否稳定。

## 课后作业

提交：

- 聚类代码
- K 值选择图
- PCA 可视化图
- 聚类画像表
- 结果解释

## 验收标准

| 项目 | 标准 |
| --- | --- |
| 标准化 | 说明为什么要标准化 |
| K 值 | 有选择依据 |
| 可视化 | 聚类图清楚 |
| 解释 | 能说明每一类可能含义 |
