# 第 3 讲：数据导入、清洗与基础可视化

## 课程定位

这一讲主入口是 [lesson-03-data-cleaning-visualization.ipynb](../../../notebooks/lesson-03-data-cleaning-visualization.ipynb)，目标是让学员跑通最小数据处理链。早期 `m_Sai_02_1.mlx` 仅作为迁移参考保留。

## 课堂目标

- 从 CSV 或 Excel 导入数据
- 识别并处理缺失值
- 计算描述性统计
- 绘制基础图表
- 保存清洗后的数据

## 核心概念

- 数据导入
- 缺失值处理
- 描述统计
- 直方图、箱线图、散点图
- 清洗结果保存

## 课堂演示

基于 Notebook 演示：

- 模拟糖尿病数据集
- 导入和清洗数据
- 生成基础可视化
- 保存清洗结果

## 课堂练习

学员完成一份小数据的清洗流程：

- 删除或填补缺失值
- 输出均值、标准差、最小值、最大值
- 生成 3 张图

## 课后作业

提交：

- 清洗后的数据文件
- 描述统计表
- 3 张图
- 200 字以内数据质量说明

## 验收标准

| 项目 | 标准 |
| --- | --- |
| 导入 | 数据能正确读取 |
| 清洗 | 缺失值处理有说明 |
| 统计 | 描述统计完整 |
| 图表 | 图表可读、标题清楚 |

## 后续建设建议

建立 `outputs/figures/` 和 `outputs/tables/`，让本讲结果自动保存。
