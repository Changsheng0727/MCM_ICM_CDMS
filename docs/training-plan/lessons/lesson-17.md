# 第 17 讲：综合模拟赛 I：读题、建模与代码

## 课程定位

这一讲开始综合模拟赛，让学员在压缩时间内完成读题、建模和代码实现。

## 课堂目标

- 在限定时间内完成题目拆解
- 确定数据处理和模型路线
- 分工完成代码和图表
- 形成初版结果

## 核心概念

- 快速读题
- 模型路线选择
- 文件分工
- 结果产出
- 进度管理

## 课堂安排

| 时间 | 内容 |
| --- | --- |
| 0-20 分钟 | 发布模拟题和数据 |
| 20-50 分钟 | 小组读题和拆解 |
| 50-90 分钟 | 确定模型路线 |
| 90-150 分钟 | 数据处理和代码实现 |
| 150-180 分钟 | 初版结果汇报 |

## 课堂练习

小组完成：

- 子问题拆解
- 数据处理方案
- 模型选择
- 代码初版
- 图表初版

## 课后作业

继续完成模拟赛报告初稿：

- 问题分析
- 模型建立
- 初步求解结果
- 图表和表格

## 验收标准

| 项目 | 标准 |
| --- | --- |
| 拆题 | 子问题清晰 |
| 模型 | 路线合理 |
| 代码 | 能产生初步结果 |
| 图表 | 能支撑结论 |

## 后续建设建议

准备 1 套难度适中的模拟赛题和配套数据。
