# 第 7 讲：综合评价模型

## 课程定位

这一讲补齐美赛高频评价类题型，训练学员建立指标体系和综合排名。

## 课堂目标

- 建立评价指标体系
- 理解正向、负向和适中型指标
- 掌握熵权法和 TOPSIS
- 理解 AHP 的主观权重思想

## 核心概念

- 指标标准化
- 权重
- 熵权法
- TOPSIS
- 一致性检验

## 课堂演示

给一份城市、学校或方案评价数据，演示：

- 指标预处理
- 权重计算
- 综合得分
- 排名与敏感性讨论

## 课堂练习

学员为一个评价问题设计指标体系，并完成 TOPSIS 排名。

## 课后作业

提交：

- 指标体系表
- 权重表
- 综合排名表
- 结果解释段落

## 验收标准

| 项目 | 标准 |
| --- | --- |
| 指标体系 | 指标与问题目标匹配 |
| 标准化 | 正负向处理正确 |
| 权重 | 权重来源说明清楚 |
| 解释 | 排名结果有实际含义 |
