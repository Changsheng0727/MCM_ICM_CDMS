# 第 15 讲：论文结构、数学表达与 LaTeX

## 课程定位

这一讲把模型训练接到论文写作，避免代码和论文脱节。

## 课堂目标

- 掌握美赛论文基本结构
- 能写问题重述、问题分析、模型假设
- 能把模型写成变量、目标、约束和算法流程
- 能使用 LaTeX 编写公式和表格

## 核心概念

- 论文结构
- 模型假设
- 符号说明
- 数学表达
- LaTeX

## 课堂演示

把一个线性规划案例写成论文段落：

- 符号定义
- 目标函数
- 约束条件
- 求解流程
- 结果引用

## 课堂练习

学员将前面某个模型写成“模型建立”小节。

## 课后作业

提交：

- 问题分析段落
- 模型假设
- 符号说明表
- 模型建立段落

## 验收标准

| 项目 | 标准 |
| --- | --- |
| 结构 | 小节完整 |
| 公式 | 符号一致 |
| 表达 | 不把结果写进模型建立 |
| 可读性 | 读者能复现模型逻辑 |

## 后续建设建议

补充一套 LaTeX 论文模板和优秀论文片段拆解。
