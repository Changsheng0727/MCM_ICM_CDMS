# 第 18 讲：综合模拟赛 II：论文打磨与复盘

## 课程定位

这一讲完成模拟赛闭环，重点训练论文整合、结果解释和复盘能力。

## 课堂目标

- 将模型结果整合成 4-6 页短报告
- 修正图表、公式和文字解释
- 检查数值结论是否有证据支撑
- 完成赛后复盘和队内角色评估

## 核心概念

- 报告整合
- 结果证据链
- 模型检验
- 论文打磨
- 赛后复盘

## 课堂安排

| 时间 | 内容 |
| --- | --- |
| 0-30 分钟 | 各组展示初稿 |
| 30-70 分钟 | 图表和结果讲评 |
| 70-120 分钟 | 报告集中修改 |
| 120-160 分钟 | 最终汇报 |
| 160-180 分钟 | 复盘和角色评估 |

## 课堂练习

小组完成：

- 摘要或结论修改
- 图表解释补齐
- 模型局限和改进补齐
- 提交最终短报告

## 课后作业

提交最终材料：

- 4-6 页短报告
- 代码和输出文件
- 小组复盘表
- 个人角色自评

## 验收标准

| 项目 | 标准 |
| --- | --- |
| 报告 | 结构完整 |
| 证据 | 数值结论有代码或图表支撑 |
| 表达 | 图表解释自然 |
| 复盘 | 能指出下次改进方向 |

## 后续建设建议

将模拟赛优秀作业整理为仓库示例。
