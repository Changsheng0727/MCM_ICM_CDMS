# 第 13 讲：仿真模型与蒙特卡洛

## 课程定位

这一讲训练学员处理不确定性和复杂系统问题。

## 课堂目标

- 理解仿真模型适用场景
- 掌握蒙特卡洛模拟基本流程
- 能输出分布、均值、方差和置信区间
- 能说明仿真次数对结果稳定性的影响

## 核心概念

- 随机变量
- 蒙特卡洛
- 置信区间
- 排队仿真
- 稳定性

## 课堂演示

演示一个需求不确定或排队等待问题：

- 设定随机分布
- 多次模拟
- 输出结果分布
- 做稳定性分析

## 课堂练习

学员模拟一个随机需求下的资源配置问题。

## 课后作业

提交：

- 仿真假设
- 仿真代码
- 结果分布图
- 置信区间和结论

## 验收标准

| 项目 | 标准 |
| --- | --- |
| 假设 | 随机设定有依据 |
| 次数 | 仿真次数足够 |
| 输出 | 有分布和统计量 |
| 解释 | 能说明不确定性影响 |

## 后续建设建议

补充元胞自动机或排队论小案例。
