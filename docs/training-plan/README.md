# MCM/ICM CDMS 培训项目计划

> 仓库入口：[MCM_ICM_CDMS](../../README.md)  
> 适用对象：准备参加 MCM/ICM、需要用 MATLAB 快速完成数据处理、建模、可视化、求解和论文结果支撑的队员。

## 一句话总线

这套培训不是单纯讲 MATLAB 语法，而是把学生训练成能完成一条比赛链路的人：

```text
读题与数据理解 -> 数据清洗与可视化 -> 基础模型 -> 优化模型 -> 结果解释 -> 论文支撑
```

## 推荐阅读顺序

1. 先读 [00-project-backbone.md](00-project-backbone.md)，明确培训目标和课程主干。
2. 再读 [05-roadmap-and-cadence.md](05-roadmap-and-cadence.md)，选择当前培训路线。
3. 授课前看 [01-course-delivery-track.md](01-course-delivery-track.md)，确认每讲交付物。
4. 开发课件时看 [03-matlab-reproducibility-track.md](03-matlab-reproducibility-track.md)，保证代码可复现。
5. 布置作业时看 [04-student-assessment-track.md](04-student-assessment-track.md)，保证学员有真实产出。
