# MCM/ICM CDMS 培训项目计划

> 仓库入口：[MCM_ICM_CDMS](../../README.md)  
> 适用对象：准备参加 MCM/ICM、需要用 Python/Jupyter 快速完成数据处理、建模、可视化、求解和论文结果支撑的队员。

## 一句话总线

这套培训不是单纯讲某个软件语法，而是把学生训练成能完成一条比赛链路的人：

```text
读题与数据理解 -> 数据清洗与可视化 -> 基础模型 -> 优化模型 -> 结果解释 -> 论文支撑
```

## 18 讲扩展方案

完整 18 讲规划见：[06-18-lesson-syllabus.md](06-18-lesson-syllabus.md)。

每一讲的课程设计文件见：[lessons/README.md](lessons/README.md)。

当前 18 讲主线为：

```text
竞赛理解 -> Python 数据处理 -> 基础模型 -> 预测与评价 -> 优化与仿真 -> 论文与模拟赛
```

其中第 1-18 讲已经有对应 Python/Jupyter Notebook；原 `.mlx` 文件只作为早期课件和迁移参考保留。

## 推荐阅读顺序

1. 先读 [00-project-backbone.md](00-project-backbone.md)，明确培训目标和课程主干。
2. 再读 [06-18-lesson-syllabus.md](06-18-lesson-syllabus.md)，确认 18 讲完整路线。
3. 打开 [../../notebooks/README.md](../../notebooks/README.md)，运行对应 Notebook。
4. 然后读 [05-roadmap-and-cadence.md](05-roadmap-and-cadence.md)，选择当前建设路线。
5. 授课前看 [01-course-delivery-track.md](01-course-delivery-track.md)，确认每讲交付物。
6. 开发课件时看 [03-matlab-reproducibility-track.md](03-matlab-reproducibility-track.md)，保证 Python/Jupyter 代码可复现。
7. 布置作业时看 [04-student-assessment-track.md](04-student-assessment-track.md)，保证学员有真实产出。
