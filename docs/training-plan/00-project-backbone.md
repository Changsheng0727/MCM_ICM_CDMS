# 00 项目背景与课程主干

## 项目定位

`MCM_ICM_CDMS` 是一个美赛培训项目仓库。当前内容以 MATLAB Live Script 为核心，已经覆盖：

- MATLAB 数据类型与数据结构
- 数据导入、清洗、筛选、描述统计和可视化
- 线性回归、K-means 聚类、PCA 和模型评估
- 线性规划、整数规划、非线性规划和多目标优化

它后续最应该补强的不是“再堆更多算法”，而是把已有课件串成一套可训练、可复现、可验收的培训体系。

## 培训对象

默认学员画像：

- 有基础数学课背景，但不熟悉完整建模竞赛流程
- 会一点编程，MATLAB 不一定熟
- 能理解模型公式，但不知道如何从题目和数据落到代码
- 最终目标是能在 MCM/ICM 中承担数据处理、建模、结果解释或论文支撑任务

## 总目标

培训结束后，学员至少应该具备：

1. 读懂题目并拆分子问题的能力
2. 用 MATLAB 完成数据导入、清洗、统计和可视化的能力
3. 根据问题类型选择基础模型的能力
4. 实现回归、聚类、PCA、优化模型的基础能力
5. 把模型结果转成图表和论文解释的能力

## 当前最重要的建设方向

### P0：把已有课件变成可交付课程

每讲都应该明确：

- 学员课前要会什么
- 课堂要跑通什么
- 课后要交什么
- 如何判断学员真的掌握了

当前优先把课程扩展到 [18 讲体系](06-18-lesson-syllabus.md)，并为每讲维护独立设计文件：[lessons/README.md](lessons/README.md)。

### P1：把代码改造成可复现实验链路

Live Script 适合教学，但比赛时还需要：

- 原始数据入口
- 清洗脚本
- 建模脚本
- 图表输出
- 结果说明

### P2：把模型结果接到论文表达

美赛不是代码比赛。每个模型都要能回答：

- 为什么选这个模型
- 输入输出是什么
- 约束或假设是什么
- 结果说明了什么
- 局限在哪里

## 不建议现在做的事

- 只增加算法，不增加作业和验收
- 只讲 MATLAB 操作，不讲建模问题拆解
- 只展示运行结果，不训练学生解释结果
- 只保留 Live Script，不沉淀可复现脚本和输出文件
