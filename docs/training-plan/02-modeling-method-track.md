# 02 建模方法线

## 这条线解决什么

建模方法线关注的是：学员能不能把美赛问题转成合适的模型，而不是只会调用 MATLAB 函数。

核心判断链：

```text
问题类型 -> 数据结构 -> 模型选择 -> 结果验证 -> 论文解释
```

## 第 3 讲方法重点

### 线性回归

学员必须能说明：

- 自变量和因变量是什么
- 为什么可以尝试线性关系
- 拟合效果如何判断
- 残差是否暴露模型问题
- 回归结果如何支撑结论

最低练习：

- 完成一元线性回归
- 完成多元线性回归
- 输出拟合图和残差图
- 写一段 150-250 字结果解释

### K-means 聚类

学员必须能说明：

- 聚类特征为什么要标准化
- K 值如何选择
- 聚类中心代表什么
- 聚类结果是否有实际含义

最低练习：

- 完成标准化
- 用肘部法选择 K
- 生成聚类散点图或 PCA 可视化图
- 输出每类特征均值表

### PCA

学员必须能说明：

- 主成分解释方差是什么
- 为什么要降维
- 降维后可视化是否保留主要结构
- PCA 结果不能被过度解释成因果关系

最低练习：

- 输出解释方差比例
- 绘制前两个主成分散点图
- 解释前两个主成分大概表达什么

## 第 4 讲方法重点

### 线性规划

学员必须能写清：

- 决策变量
- 目标函数
- 不等式约束
- 等式约束
- 变量上下界
- 求解结果和业务解释

最低练习：

- 用 `linprog` 求解一个生产计划问题
- 画出二维可行域
- 说明最优解为什么合理

### 整数规划和 0-1 规划

学员必须理解：

- 哪些变量必须是整数
- 哪些变量只表示选或不选
- 连续松弛和整数解可能不同

最低练习：

- 用 `intlinprog` 求解一个选择或分配问题
- 说明整数约束带来的变化

## 后续建议补充模型

| 优先级 | 模型 | 原因 |
| --- | --- | --- |
| P0 | 评价模型：熵权法、TOPSIS、层次分析法 | 美赛常见，容易形成完整子问题 |
| P0 | 时间序列预测：ARIMA、指数平滑、灰色预测 | 很多题有预测需求 |
| P1 | 分类模型：逻辑回归、决策树、随机森林 | 对有标签数据很实用 |
| P1 | 网络模型：最短路、最大流、图指标 | 常见于交通、传播、物流问题 |
| P2 | 仿真模型：蒙特卡洛、排队论、元胞自动机 | 用于复杂系统和不确定性 |

## 模型讲解底线

每个模型都必须配套：

- 一个实际问题
- 一个最小数据集
- 一段可运行代码
- 一张图或一张表
- 一段论文式解释
